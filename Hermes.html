Projeto Overview

Hermes é uma plataforma pensada e criada por mim para solucionar o problema de uma empresa, onde a disponibilização dos dashboards era feita de forma avulsa, em pastas, drives e e-mails, o que acabava comprometendo o controle e o acompanhamento das informações. A plataforma oferece um ambiente centralizado para clientes e administradores, além de seguir regras de controle e disponibilizar dados para análise dos fluxos e relatórios relacionados.

Sobre a plataforma, ela possui a aba de Dashboard, já com alguns indicadores e imagens prévias dos relatórios para download, podendo ser deletada se for necessário. Cada acesso é contabilizado e o download também, com os dados armazenados para estudos posteriores, como por exemplo, em Power BI, permitindo avaliar relatórios mais acessados ou não utilizados, facilitando ajustes, melhorias ou até a inativação de partes dos analistas.

A plataforma também possui a aba Tabela, que contém os mesmos dados, mas no formato de tabela. Esse formato é uma opção para analistas que preferem consultar os relatórios pelo nome, e assim fazer análises rápidas e acompanhar o histórico dos meses anteriores, pois, no principal, só é possível ver o mês atual, evitando confusões.

Há também um Input de Excel, onde o usuário escolhe o ano, o mês, o grupo e o nome do relatório para enviar. Quando a seleção é feita, o sistema dispara um e-mail para os contatos pré-definidos, informando a disponibilidade do relatório na ferramenta.

Adicionalmente, existe um Cadastro de Relatórios, onde o usuário insere os dados relacionados ao relatório, para que ele apareça na aba de Input de Excel e seja possível o envio dos e-mails previamente mencionados.

Por fim, há um Grupo de E-mails, que se refere a um serviço automatizado que possuímos, onde, todos os dias, às 12h e 18h, o sistema envia disparos para e-mails específicos de cada grupo, informando sobre os relatórios atualizados até o momento. Nessa aba, o usuário pode cadastrar novos e-mails para grupos específicos, ajustando a distribuição das informações conforme necessário.
