Projeto Overview

O bot foi desenvolvido para automatizar um processo que antes exigia uma equipe dedicada em uma empresa. Ao acessar a plataforma, o bot inicia sua operação realizando o login e aplicando os filtros necessários com agilidade e precisão. Utilizando Python, Selenium e comandos JavaScript executados pela função execute do Python, ele simula a interação humana e manipula o formulário HTML da página para capturar os dados dos campos apresentados.

O bot repete a ação com filtros diferentes, cerca de 200 vezes, gerando 200 bases individualmente. Após isso, aguarda a plataforma gerar as bases de dados requeridas e inicia a extração. O bot realiza o download das bases, utilizando Pandas para limpar, organizar e formatar as informações conforme as necessidades da empresa.

Além disso, o bot identifica padrões e realiza análises em tempo real com Scikit-learn, aplicando Machine Learning para detectar falhas ou padrões inesperados, otimizando ainda mais o processo de extração. Após preparar as bases, o bot consolida as informações e as envia para o banco de dados PostgreSQL.

Durante todo o processo, o bot é orquestrado e monitorado pela ferramenta Prefect, garantindo a execução sincronizada e eficiente de cada etapa. Os dados gerados são utilizados pelos especialistas para a criação de dashboards e estudos no Power BI.
